## Installation

```bash
$ npm install ou
$ yarn
```

## Running the app

```bash
# watch mode
$ npm run start:dev --watch ou
$ yarn start:dev --watch

# production mode
```

## Test

```bash
$ npm run test:cov ou
$ yarn test:cov
```
